1. 시스템 개요

목표: 한국 주식(KOSPI/KOSDAQ) 대상으로 매일 자동 종목 스크리닝 + Bull/Bear 토론 기반 투자 의견 제시

핵심 원칙: 스코어링/판정은 deterministic, LLM은 분석 메모 + 방향 판정만

참고: TradingAgents (48.5k stars)의 Bull/Bear debate discipline 차용

Phase 1 스크리닝 퍼널

SQL 기반 1차 필터

  • 데이터 소스: analytics_stock_daily + stock_info + analytics_valuation + consensus_estimates
  • 유동성 필터: KOSPI 시총 5,000억+ / KOSDAQ 시총 2,000억+, 거래대금 10억+
  • 추격매수 방지: 당일 +5% 초과 급등주 제외
  • 수급 데이터: 최근 5일 외국인/기관 연속 순매수일수, 누적금액
  • 컨센서스: consensus_estimates 별도 JOIN (목표가, 투자의견)

품질 필터 (scorer 단계)

필터기준목적
최소 수익률목표가 / 현재가 ≥ 1.10 (+10%)수익성 없는 추천 배제

Bull/Bear 토론 → Manager 판정 → 등급 결정

Bull Researcher (Sonnet, temperature=0)

  • 4개 analyst memo + Phase 1 정량 데이터 (_build_quant_summary) 입력
  • 매수 논거 작성 (200자)
  • 성장 잠재력, 기술적 반등, 수급 개선, 밸류 매력

Bear Researcher (Sonnet, temperature=0)

  • 매도 논거 + 한국 특화 반증 강제
  • 대주주 리스크, CB/BW/유상증자
  • 잠정실적 쇼크, DART 공시 리스크
  • 외국인 이탈, 프로그램 매도, 커버리지 부족

7. 4카테고리 스코어링

pie title 스코어 가중치 (100점 만점) "기술적" : 40 "수급" : 25 "밸류에이션" : 20 "매크로" : 15

기술적 (40점)

  • RSI 과매도: 0~10점
  • MACD 양전환: 0~10점
  • 볼린저밴드 위치: 0~5점
  • 추세 (MA 기반): 0~5점
  • 거래량 비율: 0~5점
  • 52주 위치: 0~5점

수급 (25점)

  • 외국인 순매수: 0~10점
  • 기관 순매수: 0~8점
  • 외국인+기관 동반: +4점
  • 5일 누적 순매수: 0~3점

밸류에이션 (20점)

  • PER 매력도: 0~6점
  • PBR 저평가: 0~4점
  • ROE 수준: 0~4점
  • 컨센서스 괴리율: 0~6점

매크로 (15점)

  • VIX 수준: ±5점
  • KOSPI 등락: ±3점
  • 원/달러 환율: ±2점
  • 중립 시작 (7.5점)
